# ozymandias
Read a random poem from the command line. Scrapped from https://allpoetry.com.

## Requirements
1. [bs4](https://pypi.python.org/pypi/beautifulsoup4)

## Run instructions
1. Clone the repo
2. Run `python poem.py` to read a random poem from the command line
